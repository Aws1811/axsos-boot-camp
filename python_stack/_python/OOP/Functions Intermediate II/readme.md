# Functions Intermediate II

## Objectives
- Practice writing functions and looping over dictionaries
- Achieve a better understanding of how to traverse through a list of dictionaries or through a dictionary of lists

---

## Tasks

### 1. Update Values in Dictionaries and Lists
- Change the value 10 in x to 15 → x becomes [[5,2,3],[15,8,9]]
- Change last_name of first student from 'Jordan' to 'Bryant'
- In sports_directory, change 'Messi' to 'Andres'
- Change value 20 to 30 in z

---

### 2. Iterate Through a List of Dictionaries

Create a function `iterateDictionary(some_list)` that loops through each dictionary in the list and prints each key and its value.

---

### 3. Get Values From a List of Dictionaries

Create a function `iterateDictionary2(key_name, some_list)` that prints the value of a specific key from each dictionary in the list.

---

### 4. Iterate Through a Dictionary with List Values

Create a function `printInfo(some_dict)` that prints the name of each key in uppercase along with its list size, then prints each item in the list.
