# MathDojo

## Preview

this is training on using the TDD

## Run the app

```
python math_dojo.py
```

## Built With

- Python — OOP & Unit Testing

## Features

- `add()` — adds one or more numbers to the result, supports method chaining
- `subtract()` — subtracts one or more numbers from the result, supports method chaining
- `isPositive()` — returns True if all given numbers are positive, False otherwise
- Unit tests using `unittest` — tests chaining result and isPositive behavior
