# Store & Products

this assignemnt is a training for OOP with connect them while in separate file 

## Run the app

```
python test_cases.py
```

## Built With

- Python — OOP

## Features

- `product.py` — Product class with name, price, and category; supports price updates
- `store.py` — Store class that holds a list of products; supports adding, selling, inflation, and clearance
- `test_cases.py` — creates a store, adds products, applies inflation and clearance, then sells all items
