# Python For Loops – Basic Algorithms
 
A beginner Python assignment practicing `for` loops and basic logic through 6 small challenges.
 
---
 
 
## What Each Task Does
 
### 1. Basic
Loops through every integer from 0 to 150 and prints each one. This is the simplest use of a `for` loop with a range.
 
### 2. Multiples of Five
Loops from 5 to 1,000 and prints only numbers divisible by 5. This is done by setting the range to step by 5, so it skips everything in between.
 
### 3. Counting, the Dojo Way
Counts from 1 to 100, but with two exceptions — if a number is divisible by 10, it prints `"Coding Dojo"` instead, and if divisible by 5 (but not 10), it prints `"Coding"`. The order of the conditions matters here.
 
### 4. Whoa. That Sucker's Huge
Adds up every odd integer from 0 to 500,000 and prints the total at the end. Instead of printing inside the loop, a running total is kept and only shown once the loop finishes.
 
### 5. Countdown by Fours
Starts at 2018 and counts **down** by 4 each step, printing only positive numbers. This uses a negative step in the range to go backwards.
 
### 6. Flexible Counter
Uses three variables — `lowNum`, `highNum`, and `mult` — and prints only the numbers between `lowNum` and `highNum` that are divisible by `mult`. This makes the loop reusable just by changing the variables.
