# Python String Concatenation – Hello World
 
A beginner Python assignment practicing different ways to combine and print strings.
 
---
 
## What This Assignment Covers
 
### 1. Printing with a Comma
Using a comma inside `print()` automatically adds a space between the values. It works with both strings and numbers without any conversion needed.
 
### 2. Printing with `+`
The `+` operator joins strings directly with no extra spaces. If you use it with a number, you must convert it to a string first using `str()`, otherwise Python throws an error.
 
### 3. Using `.format()`
The `.format()` method lets you write a sentence with `{}` as placeholders, then fill them in with variables. It keeps the code readable and clean.
 
### 4. Using f-strings
F-strings are the modern way to embed variables inside a string. You put an `f` before the quotes and wrap variables in `{}` directly inside the text.
 
---
 
## Key Differences
 
| Method | Auto spacing | Works with numbers | 
|--------|-------------|-------------------|
| Comma `,` | ✅ Yes | ✅ Yes |
| `+` operator | ❌ No | ❌ Needs `str()` |
| `.format()` | ✅ Yes | ✅ Yes |
| f-string | ✅ Yes | ✅ Yes |
