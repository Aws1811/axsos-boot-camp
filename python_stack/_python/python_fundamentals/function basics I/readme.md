Practice predicting the output of 15 Python function snippets before running them.

## Concepts Covered
 
**Return vs. Print** — returning a value and printing it are not the same thing.
 
**Scope** — variables inside a function stay inside it and don't affect the outside.
 
**Using return values** — a returned value can be stored, printed, or passed into another function.
 
**Conditional returns** — a function can return different values depending on the input.
 
**Chained calls** — a function call can be used directly as an argument inside another call.
