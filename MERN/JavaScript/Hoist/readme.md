# Hoist


## Run the app
Open `script.js` in your browser console or run it with Node.js:
```
node script.js
```
Or by the extension in the vs code which is named code runner.

## Built With
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) — programming language

## Features
- Rewrite JavaScript code the way the interpreter sees it using hoisting rules
- Predict the output of each code block before running it
- Understand how `var` declarations are hoisted to the top of their scope
- Understand the difference between global and function scope
- Identify hoisting errors such as calling a variable assigned function before its assignment
- Build dojo objects using ES6 `const` and apply conditional logic to set their properties
