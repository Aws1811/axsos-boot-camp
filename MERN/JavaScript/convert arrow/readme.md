# Arrow Functions

## Preview
No photos provided for this assignment.

## Run the app
Open `index.html` in your browser:
```
double click index.html
```

## Built With
- [HTML](https://developer.mozilla.org/en-US/docs/Web/HTML) — structure
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) — interactivity using arrow functions

## Features
- Change the background color of a paragraph to blue on button click
- Show a popup alert displaying the text from an input field
- Change the page background to red when hovering over a specific word and revert it on mouse out
- Change the background color of a paragraph to any color typed in an input field on click
