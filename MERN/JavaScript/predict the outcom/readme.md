# Destructuring


## Run the app
Open `script.js` in your browser console or run it with Node.js:
```
node script.js
```

## Built With
- [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) — programming language

## Features
- Destructure arrays to extract specific elements by position
- Destructure objects to extract values into renamed variables
- Predict outputs when destructuring conflicts with existing variable names
- Combine array and object destructuring to access nested values
- Understand how skipping indexes works in array destructuring
