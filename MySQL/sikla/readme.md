# 🎬 Sakila SQL Assignment

Practice SQL queries using the **Sakila** sample database — covers JOINs, filtering, and working with relational data.

---

## 📦 Database
MySQL's built-in [Sakila Database](https://dev.mysql.com/doc/sakila/en/) — a sample database based on a DVD rental store.

---

## 📝 Queries

| # | Description |
|---|-------------|
| 1 | Get all customers in `city_id = 312` |
| 2 | Get all comedy films |
| 3 | Get all films joined by `actor_id = 5` |
| 4 | Get customers in `store_id = 1` within specific cities |
| 5 | Get G-rated films with "Behind the Scenes", joined by `actor_id = 15` |
| 6 | Get all actors in `film_id = 369` |
| 7 | Get all drama films with rental rate of `2.99` |
| 8 | Get all action films joined by **Sandra Kilmer** |

---

## 💡 Concepts Used

- `JOIN` /` ON `
- `WHERE` with multiple conditions
- `LIKE` for pattern matching
- `CONCAT()` for combining columns
- Filtering by category, rating, and actor
