# 👥 Friendships

A simple SQL assignment that simulates a friendships system — like a mini social network.

---

## 🎯 What's it about?

We have users, and each user can be friends with other users. The goal is to build the database, add the relationships, and write queries to explore the data.

---

## 📦 Tables

Two tables — `users` to store people, and `friendships` to store who is friends with who.

---

## 🧠 What you'll practice

- Creating databases and tables
- Inserting and managing data
- Writing JOIN queries
- Self joins — joining the same table twice
- Aggregations and sorting

---

## 🥷 Ninja Challenges

- Who has the most friends?
- Return friends in alphabetical order
- Count all friendships in the system
