# Users Schema ERD

A database design built using MySQL Workbench.

## Schema
**users_schema**

## Tables
- **users** - stores user info (name, email, timestamps)
## Columns
- `user-id` INT (PK)
- `first-name` VARCHAR(45)
- `last-name` VARCHAR(45)
- `email` VARCHAR(45)
- `created_at` VARCHAR(45)
- `updated_at` VARCHAR(45)

## Tool Used
MySQL Workbench
