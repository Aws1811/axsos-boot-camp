## Run the app
```
# 1. navigate to the project folder
cd Java\java fundamentals\java setup\java cafe\javaCafe\src\javaCafe

# 2. compile both files
javac CafeUtil.java Main.java

# 3. run the program
java javaCafe.Main
```
Then follow the console prompts to interact with the app.

## Built With
- [Java](https://www.java.com/) — programming language

## Features
- Calculate and display the sum of numbers from 1 to a user-inputted goal
- Calculate the total price of an order from an array of item prices
- Display a numbered menu list from an ArrayList of menu items
- Add a new customer by name and inform them how many people are ahead in the queue
