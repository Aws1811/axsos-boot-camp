# Numbers Game

## Preview
No photos provided for this assignment.

## Run the app
```
# 1. navigate to the source folder
cd src

# 2. compile the file
javac NumbersGame.java

# 3. run the program
java NumbersGame
```
Then follow the console prompts to play the game.

## Built With
- [Java](https://www.java.com/) — programming language

## Features
- Randomly generate a number between 0 and 10 for the player to guess
- Track the number of attempts and warn the player as attempts run low
- Allow the player to quit anytime by typing "q"
- Validate guesses and handle invalid (non-numeric) input gracefully
- Offer the player a chance to start a new game after running out of attempts
