# Cafe Java

## Preview
No photos provided for this assignment.

## Run the app
```
# 1. navigate to the source folder
cd src

# 2. compile the program
javac com/axsos/cafeJava/CafeJava.java

# 3. run the program
java com.axsos.cafeJava.CafeJava
```
Then check your console for the order details.

## Built With
- [Java](https://www.java.com/) — programming language

## Features
- Display a welcome greeting for each customer
- Check each customer's order status and print whether it is ready or pending
- Calculate and display the total price for an order, including multi-item orders
- Adjust a customer's total to correct a billing mistake
