# List Exception Demo

## Preview
No photos provided for this assignment.

## Run the app
```
# 1. navigate to the project folder
cd src\com.axsos.list

# 2. compile the file
javac ListExceptionDemo.java

# 3. run the program
java com.axsos.list.ListExceptionDemo
```
Then check your console for the casting results.

## Built With
- [Java](https://www.java.com/) — programming language

## Features
- Store mixed data types (Strings and Integers) in an ArrayList
- Attempt to cast each list item to an Integer
- Catch and handle ClassCastException when a cast fails
- Print the index and value of items that fail to cast
