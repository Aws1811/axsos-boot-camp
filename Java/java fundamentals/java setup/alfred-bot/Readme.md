# Alfred Quotes

## Preview
No photos provided for this assignment.

## Run the app
```
# 1. navigate to the project folder
cd src\com\axsos\alfredo

# 2. compile both files
javac AlfredQuotes.java AlfredTest.java

# 3. run the program
java com.axsos.alfredo.AlfredTest
```
Then check your console for Alfred's responses.

## Built With
- [Java](https://www.java.com/) — programming language

## Features
- Display a basic greeting from Alfred
- Display a personalized greeting using a guest's name
- Announce the current date and time
- Respond differently based on whether a conversation mentions "Alexis", "Alfred", or neither
