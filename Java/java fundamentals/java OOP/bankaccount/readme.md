## Run the app
```
# 1. navigate to the project folder
cd src\com.axsos.bank

# 2. compile all files
javac BankAccount.java BankTest.java

# 3. run the program
java com.axsos.bank.BankTest
```
Then check your console for the account balances.

## Built With
- [Java](https://www.java.com/) — programming language

## Features
- Create multiple bank accounts, each with separate checking and savings balances
- Deposit money into a checking or savings account
- Withdraw money from a checking or savings account, with insufficient funds protection
- Display the checking, savings, and total balance for an individual account
- Track and display the total number of accounts created and the combined total money across all accounts using static fields
