## Run the app
```
# 1. navigate to the project folder
cd src\com.axsos.barista

# 2. compile all files
javac Items.java Order.java TestBara.java

# 3. run the program
java com.axsos.barista.TestBara
```
Then check your console for the order details.

## Built With
- [Java](https://www.java.com/) — programming language

## Features
- Create menu items with a private name and price, accessed through getters and setters
- Create orders for guests (default name) or named customers using multiple constructors
- Add multiple items to an order and track them in a list
- Mark orders as ready or not ready and display the corresponding status message
- Calculate and display each order's total price
- Display a full order summary including customer name, items, prices, and total
