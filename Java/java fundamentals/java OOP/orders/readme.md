## Run the app
```
# 1. navigate to the project folder
cd src\com.axsos.items

# 2. compile all files
javac Items.java Order.java TestOrder.java

# 3. run the program
java com.axsos.items.TestOrder
```
Then check your console for the order details.

## Built With
- [Java](https://www.java.com/) — programming language

## Features
- Create menu items with a name and price
- Create customer orders with a name, total, ready status, and a list of items
- Add items to an order and update its running total
- Mark orders as ready
- Display each order's name, total, and ready status
