# Devices

## Preview
No photos provided for this assignment.

## Run the app
```
# 1. navigate to the project folder
cd src\com.axsos.oop.devices

# 2. compile all files
javac Device.java Phone.java TestDevice.java

# 3. run the program
java com.axsos.oop.devices.TestDevice
```
Then check your console for the battery updates.

## Built With
- [Java](https://www.java.com/) — programming language

## Features
- Create a base Device class with a battery level that starts at 100
- Extend Device with a Phone class using inheritance
- Drain battery by making calls or playing games
- Recharge the battery with a charging method
- Display the current battery level at any time
