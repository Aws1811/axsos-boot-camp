# Coffee App

## Preview
No photos provided for this assignment.

## Run the app
```
# 1. navigate to the project folder
cd src\com.axsos.coffeeApp

# 2. compile all files
javac Items.java Order.java CoffeeKiosk.java Main.java

# 3. run the program
java com.axsos.coffeeApp.Main
```
Then follow the console prompts to place your order.

## Built With
- [Java](https://www.java.com/) — programming language

## Features
- Add menu items with a name, price, and auto-assigned index
- Display the full menu with item index and price
- Create a new order by entering a customer name
- Interactively add items to an order by entering their menu index
- Quit adding items and finalize the order by typing "q"
- Display the completed order summary including customer name, items, and total price
