## Run the app
```
# 1. navigate to the project folder
cd Desktop\axsos\Java\spring boot\routes

# 2. build and run the Spring Boot app
./mvnw spring-boot:run
```
Then open your browser at: `http://localhost:8080/daikichi`

## Built With
- [Java](https://www.java.com/) — programming language
- [Spring Boot](https://spring.io/projects/spring-boot) — Java web framework

## Features
- Set up a Spring Boot REST controller under the `/daikichi` base route
- Return a welcome message at `/daikichi`
- Return a daily fortune message at `/daikichi/today`
- Return a tomorrow fortune message at `/daikichi/tomorrow`
