# Clock Tick Project

This assignment demonstrates a clock ticking mechanism.

It works using `setTimeout` and `setInterval`, along with `clearInterval` to control and stop the timing process.

## Features
- Simulates a ticking clock using JavaScript timing functions
- Demonstrates control over execution timing
- Includes a hidden button that triggers an unexpected behavior

## Description
The project focuses on how timing functions in JavaScript can be used to create dynamic, time-based interactions. Additionally, a hidden element is included to add a mysterious or experimental feature.
